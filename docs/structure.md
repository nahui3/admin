# Структура проекта

```
admin/
├── manage.py          # Утилита управления Django
├── install            # Скрипт установки (macOS/Linux)
├── install.bat        # Скрипт установки (Windows)
├── run                # Скрипт запуска сервера (macOS/Linux)
├── run.bat            # Скрипт запуска сервера (Windows)
├── configs/           # Конфигурационные файлы
│   ├── __init__.py
│   ├── settings.py    # Настройки проекта
│   ├── urls.py        # URL-маршруты
│   ├── wsgi.py        # WSGI конфигурация для развёртывания
│   └── asgi.py        # ASGI конфигурация для развёртывания
├── docs/              # Документация
│   ├── README.md
│   ├── install.md
│   ├── structure.md
│   ├── settings.md
│   ├── deploy.md
│   └── commands.md
├── db.sqlite3         # База данных SQLite (создаётся после migrate)
├── venv/              # Виртуальное окружение (не в git)
└── README.md          # Основная инструкция
```

## Описание файлов

### Основные файлы

- **manage.py** - Основная утилита для управления Django проектом. Используется для выполнения команд Django (migrate, runserver, createsuperuser и т.д.)

### Скрипты установки и запуска

- **install** / **install.bat** - Скрипты для автоматической установки всех зависимостей
  - Создают виртуальное окружение
  - Устанавливают Django
  - Применяют миграции базы данных
  - Предлагают создать суперпользователя
  - Bash версия для macOS/Linux, Batch версия для Windows

- **run** / **run.bat** - Скрипты для запуска сервера разработки
  - Активируют виртуальное окружение
  - Проверяют установку Django
  - Применяют миграции при необходимости
  - Запускают сервер разработки
  - Автоматически открывают админ-панель в браузере
  - Bash версия для macOS/Linux, Batch версия для Windows

### Конфигурационные файлы

- **configs/settings.py** - Основные настройки проекта (база данных, приложения, middleware, язык, часовой пояс и т.д.)
- **configs/urls.py** - Определение URL-маршрутов проекта. Содержит маршрут к админ-панели `/admin/`
- **configs/wsgi.py** - WSGI конфигурация для production развёртывания (используется с Gunicorn, uWSGI и т.д.)
- **configs/asgi.py** - ASGI конфигурация для async развёртывания (используется с Daphne и другими ASGI серверами)

### Дополнительные файлы

- **db.sqlite3** - База данных SQLite (создаётся автоматически после выполнения `migrate`)
- **venv/** - Виртуальное окружение Python (создаётся скриптом `install`, не хранится в git)
- **docs/** - Папка с подробной документацией проекта
