# Документация

Добро пожаловать в документацию! Здесь вы найдёте всю необходимую информацию для работы с проектом.

## Содержание

- [Установка и развёртывание](install.md) - Подробная инструкция по установке

## Быстрый старт

1. Клонируйте репозиторий: `git clone https://github.com/nahui3/admin.git`
2. Запустите установку: `./install`
3. Запустите сервер: `./run`
