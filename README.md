# Админ-панель NAHUI

Административная панель NAHUI на базе фреймворка Django.

## Требования

- Python 3.8 или выше
- pip (менеджер пакетов Python)

## Быстрый старт

### macOS / Linux

```bash
# Клонируйте репозиторий
git clone https://github.com/nahui3/admin.git
cd admin

# Запустите автоматическую установку
./install

# Запустите сервер (автоматически откроется в браузере)
./run
```

### Windows

```cmd
REM Клонируйте репозиторий
git clone https://github.com/nahui3/admin.git
cd admin

REM Запустите автоматическую установку
install.bat

REM Запустите сервер (автоматически откроется в браузере)
run.bat
```

Админ-панель будет доступна по адресу: http://127.0.0.1:8000/admin/
